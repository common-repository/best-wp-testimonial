== Installation ==
1. Upload `best-wp-testimonial` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste <strong>if(function_exists('bwpt_testimonial_shortcode')){bwpt_testimonial_loop();}</strong> in template code or  <strong>[BWPTTESTIMONIAL]</strong> in the post/page" to display testimonial.
